10 Iris Recognition Open Source Projects List

[Greetings, my fellow reader!]{style="font-weight: 400;"}

[I am Denis Trofimov, a team leader developer working on iris recognition project for the company Vsor Systems LLC, Moscow.]{style="font-weight: 400;"}

I have recently published this post with a full-sized table of all Iris Recognition Open Source Projects.

I have decided to reorganize information as a list to use as fast reference. I have checked all the links to be working on the publish date.

Iris Recognition Open Source Projects List:

  1. Video-based Automatic System for Iris Recognition (VASIR) (Beta Version 2.2) 2013, C++. Requires Qt Framework 4.8.2, OpenCV Library 2.3.1.
  2. [5455945/Iris_Osiris]{style="font-weight: 400;"} the OSIRIS 4.1 fork, 2017, C++.Requires OpenCV 3.0+, OpenCV 2.4.13.
  3. [Python interface to OSIRIS Iris Segmentation and Recognition software]{style="font-weight: 400;"}[(the OSIRIS 4.1 fork) 2017, Python.]{style="font-weight: 400;"}
  4. (USIT) - University of Salzburg Iris Toolkit v2  2016, C++. Requires OpenCV version 2.4.9, Python 3, scipy.
  5. (UND) IRIS RECOGNITION BASED ON HUMAN-INTREPRETABLE FEATURES [2016, Matlab.]{style="font-weight: 400;"}
  6. [Libor Masek`s ]{style="font-weight: 400;"}Dissertation Iris Recognition 2003, MATLAB.
  7. My VASIR fork to use OpenCV 3.4 and later, 2018, C++.
  8. [IrisSeg]{style="font-weight: 400;"}[[ A Fast and Robust Iris Segmentation Framework for Non-Ideal Iris Images, ]{style="font-weight: 400;"}]{style="font-weight: 400;"}2017, Matlab.
  9. IAADseg An Accurate Iris Segmentation Framework under Relaxed Imaging Constraints using Total Variation Model, [2015, Matlab.]{style="font-weight: 400;"}
  10. IrisSeg [Iris Segmentation using Geodesic Active Contours and GrabCut, 2015, Python.]{style="font-weight: 400;"}
